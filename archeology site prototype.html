ArchiVault

Precision Microclimate Control for Small Artifacts

ArchiVault is a specialized transport crate engineered to maintain stable conditions for small archaeological artifacts during movement. Our system provides comprehensive control over temperature, humidity, oxygen levels, and shock absorption while continuously monitoring and logging all environmental conditions to ensure artifact safety.

Explore Systems Technical Details

Eight Integrated Systems

Each system has a clear job, working together to protect artifacts in a sealed microclimate.

1. Control & Logging

MCU-based monitoring with BME280 and accelerometer sensors. Continuous data export to graphs for real-time condition tracking and anomaly detection.

2. Sensor Array

BME280 for temperature/humidity, MPU6050 for motion detection, analog O2 sensor (0-3.3V scaled). Strategic placement measures artifact bay air, isolated from electronics heat.

3. Gas & Oxygen Control

Nitrogen purge system with portable cylinder, regulator, and bulkhead port. Needle valve for flow control, check valve for safety, O2 probe port for sampling without opening.

4. Humidity Management

Silica gel canisters with saturation indicators, RH indicator cards for quick checks. Optional buffer packs sized to bay volume for precise RH targeting.

5. Temperature Control

Passive: insulation and gel packs. Active: Peltier module with heatsink/fan. Thermostat/relay prevents oscillation. Heat dump outside artifact bay.

6. Impact Isolation

Memory foam cradle system with spring-mounted suspension. Shock/tilt logger tags incidents in data log for complete transport history.

7. Power System

12V Li-ion power bank (200Wh capacity). Buck converter provides 5V for MCU/sensors. Fused protection with master switch for safety.

8. Artifact Bay

Sealed compartment with cable glands for sensors. Service port for sampling/adjustments. Secure straps with adequate clearance for various artifact sizes.

Key Features

🌡️ Precision Climate Control

Maintains stable temperature and humidity using BME280 sensors with active or passive control systems. Real-time monitoring prevents thermal shock and moisture damage.

🧪 Controlled Atmosphere

Nitrogen purge system reduces oxygen levels to prevent oxidation. Integrated O2 monitoring with sampling port allows verification without compromising the sealed environment.

📊 Continuous Data Logging

MCU-based system tracks all environmental parameters and motion events. Exportable graphs provide complete transport history for documentation and analysis.

🔒 Multi-Layer Protection

Memory foam cradle with spring-mounted suspension absorbs impacts. Shock logger records any incidents for complete chain-of-custody documentation.

⚡ Autonomous Operation

200Wh Li-ion battery system provides extended operation. Efficient power management with 12V to 5V conversion for sensor electronics.

🚨 Alarm System

Automatic alerts when conditions exceed safe ranges. Visual and data-logged warnings ensure immediate response to environmental threats.
